{\bf Problem \arabic{Prob}:} Suppose C is some curve from \(\left( 0, 0, 0 \right)\) to \(\left( 1, 1, 1
\right)\) and \(\mathbf{F} \left( x, y, z \right) = \left( 4 z + 4 y \right)
\mathbf{i} + \left( z + 4 x \right) \mathbf{j} + \left( y + 4 x \right)
\mathbf{k}\). Compute the line integral directly \(\int_{C} \mathbf{F} \cdot d\mathbf{r}\). ({\em Do not try to find a potential function})
